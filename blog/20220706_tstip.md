## ts tips    

### 环境配置  


```javascript
{
  "compilerOptions": {
    ...,
    "baseUrl": "../",
    // 如果`webpack/rollup`等配置了路径别名，需要在`tsconfig.json`的`compilerOptions`同时配置`path`:
    "paths":{
        "@/*":["src/*"],
        "@":["src"],
    },
    // 声明暴露这些文件下的 .d.ts （不声明这个会导致.d.ts无效）
    "include": ["src/**/*.ts", "src/**/*.d.ts", "src/**/*.tsx", "src/**/*.vue","*.d.ts"],
  },
}

```
